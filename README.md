# 🌍 Afibuy - Liberia Global Trade Intelligence Platform

**AI-Powered Trade Analytics & Smart Marketplace for Liberia & West Africa**

---

## 📋 Overview

Afibuy is a comprehensive **Streamlit-based web application** that provides Liberian importers, traders, wholesalers, and entrepreneurs with powerful trade intelligence tools. 

It combines rich data visualization, advanced Machine Learning insights, opportunity discovery, and a basic seller marketplace — all in one intuitive dashboard.

---

## ✨ Key Features

### 📊 **Trade Analytics**
- Interactive Liberia Trade Overview with key metrics
- Top importing countries and port analytics
- High Opportunity Products explorer (filtered by score)
- Liberia Export Intelligence

### 📈 **Advanced Visualizations**
- Sankey Diagram, Sunburst, Treemap, Bubble Chart, Heatmap
- 3D Scatter, Box Plots, Violin Plots, Waterfall, Funnel, Polar Charts
- Built with Plotly

### 🤖 **AI & Machine Learning Tools**
- **AI Price Predictor** (Random Forest Regressor)
- Smart Product Recommendations (Cosine Similarity)
- Market Segmentation (K-Means Clustering)
- Budget Optimizer
- Trend Forecasting & Anomaly Detection

### 🛍️ **Marketplace**
- User authentication (Register/Login/Guest)
- Add new products with image upload
- Browse product listings

---

## 🛠️ Tech Stack

- **Framework**: Streamlit
- **Data Processing**: Pandas, NumPy
- **Visualization**: Plotly Express & Graph Objects
- **ML**: scikit-learn (RandomForest, KMeans, Cosine Similarity, etc.)
- **Database**: SQLite3
- **Auth**: Custom password hashing
- **Others**: Joblib, SciPy, Pathlib
